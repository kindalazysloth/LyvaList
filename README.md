Transform your browser's new tab into a beautiful, organized workspace.

Lyvalist is a visual bookmark manager featuring drag-and-drop boards, cloud sync, and lightning-fast saving. Best of all? It includes all the premium features you'd normally pay for, completely free.



✨ Features
🎨 Visual Boards: Organize your bookmarks into clean, drag-and-drop boards rather than cluttered lists.

📑 Multiple Pages: Create separate environments for Work, Personal, and active Projects. Switch between them instantly.

⚡ Quick Save Shortcut: Found a great link? Hit Ctrl+B (Cmd+B on Mac) to save the page instantly without breaking your workflow.

☁️ Cloud Sync: Sign in with Google to keep your layout and links perfectly synced across all your devices automatically.

🕵️ Privacy Mode: Blur your bookmarks with a single click—perfect for when you are sharing your screen on a call.

📥 Seamless Import: Bring in your existing Chrome bookmarks in seconds to get started right away.


🛠️ How to Install
Since Lyvalist is not yet on the Web Store, you can install it directly from this release in just a few seconds:
1.Download the Release:Grab the zip file from the Assets section below.Scroll to the bottom of this page and download the 36MB Lyvalist.zip file.

2.Extract the Files:Don't delete this folder later!.Unzip the downloaded file to a permanent location on your computer (like your Documents folder). Note: If you delete or move this folder later, Chrome will uninstall the extension.

3.Open Chrome Extensions:Open Google Chrome, type chrome://extensions/ into your address bar, and hit Enter

4.Enable Developer Mode:Required for custom extensions.Find the Developer mode toggle in the top-right corner of the Extensions page and turn it on.

5.Load the Extension:Click the Load unpacked button that appears in the top-left corner. Select the extracted Lyvalist folder from Step 2.


Open a new tab, and your new personal dashboard is ready to go!
